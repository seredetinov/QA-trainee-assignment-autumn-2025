## Общее предусловие (Базовая структура запроса)

Составить HTTP-запрос со следующей структурой:
```
1. Протокол HTTP/1.1
2. Заголовок Host со значением qa-internship.avito.com/
3. Заголовок Content-Type со значением application/json
4. Заголовок Accept со значением application/json
```

## Тест-сьют "Получить объявления по его идентификатору"

### №1 Получение существующего объявления

**Предусловия**:

**Шаги**:

1. Указать GET в качестве метода запроса.
2. Указать эндпоинт "/api/1/item/{id}", где id - это значение любого существующего объявления.

1. Код ответа 200.

3. Значение параметра "id" в ответе совпадает с тем, который был указан в эндпоинте.

Пройден
